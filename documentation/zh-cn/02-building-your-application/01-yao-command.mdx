Yao 是一个独立的可执行文件。即使你有多个项目，每台机器或服务器也只需要安装一次。

本节介绍基本的 Yao 命令，这些命令将在后面用于构建一个简单的应用程序。快速浏览一下就足够了。

有关更详细的信息，请参阅 Yao 命令参考。

👉 **[Yao 命令参考](../references/yao)**

## 概述

Yao 命令的基本用法：

| 命令          | 描述                   | 示例                    |
| ------------- | --------------------- | ----------------------- |
| `yao start`   | 启动应用服务器         | `yao start`             |
| `yao run`     | 运行一个进程           | `yao run utils.app.Ping` |
| `yao migrate` | 迁移数据库，构建架构    | `yao migrate`           |
| `yao version` | 显示 Yao 的版本        | `yao version --all`     |

SUI 命令，用于网页构建：

| 命令           | 描述                            | 示例                      |
| -------------- | ------------------------------ | ------------------------- |
| `yao sui build`| 构建 SUI 页面                  | `yao sui build web default`|
| `yao sui watch`| 监视 SUI 页面变化，并自动重新构建 | `yao sui watch web default`|
